# C++ Debug Visualizer

A VS Code extension that makes debugging C++ code more visual and intuitive.

## Features

- Array/Vector Visualizer: see your arrays as visual blocks
- Memory Visualizer: stack and heap visualization in real time
- Pointer Explorer: understand pointer relationships graphically
- Live Variables Panel: track your variables in real time
- Call Stack Beautifier: a cleaner and colorful call stack

## Requirements

- VS Code 1.80+
- A C++ project with a launch.json debug config

## Installation

Search for `cpp-debug-visualizer` in the VS Code Marketplace and click Install.
